# DevAscDemo
Devnet Associate training 2025

Minimal demo repository for a tiny Python feature used in DevNet Associate training.

Usage
- Run the example script:

	python -m pyscripts.myscript [name]

Example

	python -m pyscripts.myscript Alice

This will print a greeting and a timestamp using the `Greeter` class in
`Feature/myfeature.py`.
